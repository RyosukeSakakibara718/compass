# 環境構築手順

## 環境起動
下記コマンドはすべてルートディレクトリで実行
### 1. 各種ライブラリインストール
``` 
npm install
```

### 2. envファイルの用意
```
cp .env.example .env
```

### 3. 開発環境の起動
```
npm run dev
```
